# 📌 Smart Task Manager using DSA

## 📖 Overview
This project is a **Task Management System** that demonstrates the practical use of **Data Structures and Algorithms (DSA)** in solving real-world problems faced by students.

Instead of a simple to-do list, this system integrates multiple DSA concepts to make task management efficient and intelligent.

---

## ❗ Problem Statement
Students often face difficulties like:
- Not knowing which task to do first  
- Unable to undo mistakes  
- Slow searching of tasks  
- No autocomplete suggestions  
- Difficulty in choosing tasks within limited time  
- Finding shortest path between locations  

---

## 💡 Solution
We developed a **Smart Task Manager** that solves each problem using a specific DSA approach.

---

## ⚙️ DSA Concepts Used

### 1. Stack (Undo/Redo)
- Implements undo and redo operations  
- Follows **LIFO (Last In First Out)**  
- Time Complexity: `O(1)`

---

### 2. Max Heap (Priority Queue)
- Automatically sorts tasks based on priority  
- Highest priority task comes first  
- Time Complexity: `O(log n)`

---

### 3. HashMap
- Fast task lookup using task ID  
- Time Complexity: `O(1)`

---

### 4. Trie (Autocomplete)
- Provides search suggestions while typing  
- Efficient prefix-based searching  

---

### 5. Dijkstra Algorithm
- Finds shortest path (e.g., Home → Lab)  
- Used for navigation problems  

---

### 6. Knapsack Algorithm
- Selects best tasks within limited time (e.g., 180 mins)  
- Optimizes productivity  

---

## 🚀 Features
- ✅ Add and manage tasks  
- ✅ Undo/Redo functionality  
- ✅ Priority-based task sorting  
- ✅ Fast search and autocomplete  
- ✅ Shortest path calculation  
- ✅ Optimal task selection
